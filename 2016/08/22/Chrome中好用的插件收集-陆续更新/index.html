Chrome中好用的插件收集(陆续更新)

Chrome
  • Chrome
  • Debian
  • Testing/Stable

简介

谷歌浏览器 Google Chrome 最大的特色之一就是拥有非常丰富的插件扩展，基本你能想得的功能都有实现，Chorme WebStore (网上应用商店) 简直就像是一个宝库，然而很多优秀的扩展却被埋没。

Chrome 插件总结

请使用 「番·羽·士·啬·工·具」下载使用下述插件

  1. Proxy SwitchyOmega：浏览器代理设置神器，前身为 Proxy SwitchySharp，可以轻松快捷地管理多个代理，并能快速地在代理之间切换。可以设置规则或使用 PAC 自动智能对指定的网站使用代理，一句话 番·羽·土·啬 必备.
  2. Google翻译:可以比较轻松的翻译网页中的内容.
  3. 表情帝:表情帝 for QQ空间——让您发表状态、回复好友时能够添加各种搞怪、卡哇伊表情,再也不用担心没有好表情了.
  4. 百度贴吧助手:百度贴吧自动签到，自定义表情，阅读魔法书，翻转帖子中图片，设置朋友昵称，自定义背景图，一键换号，大召唤系统，吧务工具扫描广告贴，等等
  5. JSON-handle:自动格式化网页上的json,android开发者应该会喜欢
  6. 二维码(QR码)生成器(QR Code Generator):可以将喜欢但是没有看完的网页的网址生成一个二维码,手机扫描就可以在路上看了.
  7. Octotree:Code tree for GitHub and GitLab
  8. 视频广告拦截:这个没有什么推荐的必要,将视频广告拦截六个字放在Chrome 网上应用店搜索就能有很多自己慢慢选择吧!
  9. RSS Feed Reader:让你的工具栏上显示一个简单RSS订阅.
  10. 阅读模式:提供与Safari阅读模式功能一致的插件，浏览文章页时候可进入友好的阅读模式，并自定义阅读功能.
  11. OneTab:节省高达95％的内存，并减轻标签页混乱现象.将打开的标签页按照打开的时间产生一个tag并很方便的管理他们.
  12. Chrome 远程桌面:通过互联网安全访问其他计算机，或允许其他用户通过互联网安全访问您的计算机。
  13. Secure Shell:终端模拟器和SSH客户端。
  14. Postman：一个相当强大好用的 REST API & HTTP 请求调试工具，它能够发送任何类型的HTTP requests (GET, HEAD, POST, PUT..)，附带任何数量的参数+ headers；支持不同的认证机制（basic, digest, OAuth），接收到的响应语法高亮（HTML，JSON或XML）；Postman 可以简化测试和调试HTTP请求，是每个 Web 开发人员都值得安装的扩展。
